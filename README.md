The React Employee Management System is a simple and interactive web application built using ReactJS.
It allows users to add, view, edit, and delete employee records efficiently. This project demonstrates practical knowledge of React state management, forms, and dynamic rendering of data in a table.

Key Features:

Add new employees with details: First Name, Last Name, Job Title, Job Role, Job Description.

Display all employees in a table format.

Edit existing employee details with the Edit button.

Remove employees from the list using the Delete button.

Reusable form for both Add and Edit operations.

Fully built using React functional components and Hooks (useState).

This project is ideal for learning CRUD operations in React and serves as a good example for front-end development portfolios.

Technologies Used

ReactJS – Front-end library

JavaScript – Core programming language

HTML/CSS – Styling and layout

VS Code – Development environment

Node.js & npm – Project setup and running
